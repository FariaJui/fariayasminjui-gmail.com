# Digital Forensics System

## Overview

Digital Forensics System is a console-based application developed in "C++" using Object-Oriented Programming (OOP). The purpose of this project is to demonstrate the basic process of handling digital evidence in a simple and organized way.
The system allows users to collect evidence, store important information, calculate file size, generate hash values, search evidence records, and create investigation reports. It also keeps track of the chain of custody to help maintain evidence integrity during the investigation process.

## Project Objectives

-> Manage digital evidence in an organized way
-> Store and retrieve evidence information
-> Generate custom hash values for basic integrity checking
-> Maintain chain of custody records
-> Generate forensic reports
-> Practice and implement OOP concepts in a real-world project
